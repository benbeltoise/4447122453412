Option C: Appli - Job Application Tracker

Setup instructions:

1. Extract the ZIP file and open the project in Visual Studio Code

2. Install Dependencies using the project terminal:
   npm install

3. Start the app using the project terminal:
   npx expo start

4. Run on emulator:
   - press "a" to open android emulator
   - press "i" to open iOS emulator
   - Or scan QR code using Expo Go on a mobile device

5. Log in using the pre-seeded demo account:
   Email: demo@student.com
   Password: demouser

6. Test out the various features
